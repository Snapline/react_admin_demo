#  `react_news_demo`

项目根据慕课网的课程进行

### 技术栈

构筑自动化工具： **Webpack**

前端框架： **React**

路由控制： **react-router**

组件库： **ant design**

网络请求： **fetch**

pc移动响应式： **react-responsive**

### 环境安装

安装依赖包：

```bash
  npm install
```

本地服务及热加载开启：

```bash
  npm run dev
```

代码打包压缩：
webpack.config.js里改为devtool: false

```bash
  webpack -p
```
样式没有细致调节，接口使用老师做好的，可能会有维护问题和跨域问题导致某些接口不能访问。
